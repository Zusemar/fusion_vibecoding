# ОМ ИД №3 — Александр Василенко — вариант 4

Генератор создаёт отдельные модели для заданий 2–6. Геометрия строится автоматически,
а материалы, контакты, нагрузки, исследования и joints настраиваются вручную в Fusion.

## Параметры варианта

- Студент: Александр Василенко
- Вариант: 4
- Материал варианта: `PBT plastic`
- Нагрузка на полку: `400 N`
- Зубья для заданий 3–4: `4`
- Зубья механизма: `4` и `8`
- Передаточное отношение: `i = z2 / z1 = 8 / 4 = 2`

## Запуск генератора

1. Открыть Fusion.
2. Перейти в `Utilities → Scripts and Add-Ins`.
3. На вкладке `Scripts` нажать зелёный `+` и выбрать папку
   `Lab3_V4_Generator` из этого проекта.
4. Выбрать `Lab3_V4_Generator` и нажать `Run`.
5. Ввести номер задания: `2`, `3`, `4`, `5` или `6`.
6. Сохранить созданный документ с отдельным именем.

Если Fusion не показывает текст на модели, имя `LABEL — Александр Василенко —
вариант 4` остаётся видимым в Browser. Для финального кадра можно также создать
`Create Sketch → Text` и написать `Александр Василенко — вариант 4`.

## Задание 2 — статический анализ полки

Генератор создаёт стену, стальную полку `1000 × 300 × 3 мм` и три реалистичных
ребристых кронштейна. Каждый кронштейн имеет широкую заднюю монтажную пластину,
горизонтальную опору, два боковых ребра, четыре стеновых анкера и два винта,
притягивающих полку. Головки и шайбы добавлены для реалистичного вида.

1. Перейти в `Simulation → New Study → Static Stress`.
2. Назначить стене, полке и стержням крепежа `Steel`.
3. Назначить всем деталям внутри сборок `BRACKET` материал `PBT plastic`.
4. Наложить `Fixed` на заднюю плоскость стены.
5. Приложить `400 N` вертикально вниз к верхней поверхности полки.
6. Настроить контакты:
   - стержни анкеров со стеной и монтажными пластинами: `Bonded`;
   - стержни винтов полки с полкой и горизонтальными опорами: `Bonded`;
   - детали внутри каждого цельного кронштейна: `Bonded`;
   - монтажные пластины со стеной вне анкеров: `Sliding`;
   - полка со стеной: `Separation`;
   - полка с горизонтальными опорами: `Sliding`.
7. Исключить из исследования компоненты с пометкой
   `EXCLUDE FROM SIMULATION`: головки и шайбы нужны только для реалистичного вида.
8. Запустить `Generate Mesh`, локально измельчить сетку около отверстий и рёбер,
   затем `Solve`.
9. Зафиксировать `Safety Factor ≥ 3` и максимальное перемещение `≤ 10 мм`.

Для оптимизации сравнить исходные три кронштейна с вариантом уменьшенной толщины
монтажной пластины, опоры или рёбер. Оставить самый дешёвый вариант, выполняющий
два ограничения.

## Задание 3 — статический стресс зубчатого зацепления

Генератор создаёт два колеса с четырьмя плавно скруглёнными зубьями. Второе
колесо развёрнуто зубом во впадину первого; между контурами оставлен зазор `1,2 мм`,
поэтому они не пересекаются.

1. Назначить обоим колёсам `Steel`.
2. Создать `Static Stress`.
3. Зафиксировать цилиндрическую поверхность отверстия колеса A.
4. Приложить момент к отверстию колеса B.
5. Между контактирующими зубьями задать `Separation`.
6. Измельчить сетку на зубьях и выполнить несколько расчётов с разным моментом.
7. Показать самый нагруженный результат и сравнение.

## Задание 4 — оптимизация формы

Заготовка также использует четыре плавно скруглённых зуба, но сделана крупнее и
толще, чтобы при Shape Optimization было достаточно материала для удаления.

1. Создать `Shape Optimization` для колеса.
2. Сохранить как `Preserve Region` отверстие и поверхности зубьев.
3. Зафиксировать отверстие, приложить момент и касательную нагрузку к зубьям.
4. Запустить расчёт и показать исходную и оптимизированную массу.

## Задание 5 — динамический удар

1. Назначить пластине `PBT plastic`, молотку `Steel`.
2. Создать `Dynamic Event Simulation`.
3. Зафиксировать края или нижнюю поверхность пластины.
4. Между бойком и пластиной задать `Separation`.
5. Между бойком и рукояткой задать `Bonded`.
6. Задать обеим частям молотка начальную скорость вниз, например `1000 mm/s`.
7. Уменьшить длительность события так, чтобы на графике был виден удар.
8. На видео остановиться в момент максимального напряжения и показать `Min/Max`.

## Задание 6 — кинематика

Короткий запуск:

1. Запусти генератор, введи `6`.
2. Правой кнопкой → `Ground`: `AXLE 1`, `AXLE 2`, `SLIDER GUIDE`.
3. `Assemble → Rigid Group`: объедини `GEAR 2 + CRANK PIN` и
   `SLIDER + SLIDER PIN`.
4. `Assemble → As-Built Joint → Revolute`: соедини колёса с осями и шатун с
   пальцами. Для ползуна создай `Slider Joint`.
5. `Assemble → Motion Link`: задай `360°` для GEAR 1 и `-180°` для GEAR 2.
6. Правой кнопкой по joint колеса 1 → `Drive Joint` → Play.

Шестерни специально разнесены на `70 мм`: это убирает пересечение упрощённых
зубьев при движении.

Расчёт для отчёта:

```text
i = z2 / z1 = 8 / 4 = 2
n2 = n1 / 2
ω2 = 2πn2 / 60
v_кривошипа = rω2
```

Например, при `n1 = 120 rpm` и радиусе кривошипа `r = 25 mm`:

```text
n2 = 60 rpm
ω2 = 2π rad/s
v_кривошипа = 0.025 × 2π = 0.157 m/s
```
